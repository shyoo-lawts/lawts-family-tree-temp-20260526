# 가계도 자동 생성기 (베타 v0.1)

법무법인 더스마트 · 채무대응팀 · 한정승인/상속포기 사건용

## 라이브 데모

배포 후 URL: `https://shyoo-lawts.github.io/lawts-family-tree-temp-20260526/`

## 핵심 기능

- 표에 가족 입력 → SVG 가계도 자동 시각화
- 민법 1000조~1009조 기반 **상속인 자동 분석** (1·2·3·4순위 + 배우자 + 대습)
- **다단계 대습상속** 자동 처리 (자녀→손자녀, 형제→조카→종손)
- 망인 ✝ / 상속인 ★ / 사망 故 / 한정·상포 캡슐 배지
- 컴팩트 모드 (큰 가족 한 화면에)
- 실무 표기 (피상속인 망 / 청구인 / 청구외 망 / 청구외)
- 자동 판정 ON/OFF 토글 (변호사 검수 모드)
- localStorage 자동 저장 · JSON 내보내기/불러오기
- PNG 저장 · PDF/A4 가로 인쇄

## 사용 흐름

1. 표에서 **망인 1명** 지정 (피상속인) + 사망 일자 입력
2. 가족 관계 추가 (배우자/자녀/형제자매 등)
3. 상태 설정 (생존/한정승인/상속포기/사망)
4. 우측 자동 분석 결과 + 가계도 SVG 확인
5. PNG 저장 또는 PDF 인쇄로 법원 제출

## 샘플 3종 내장

- **직계비속** (망인 + 배우자 + 자녀)
- **대습상속** (자녀 사망 → 손자녀)
- **형제자매 상속** (직계 없음 → 형제)

## ⚠ 주의

이 도구의 **자동 분석 결과는 참고용**입니다. 실제 사건은 반드시 담당 변호사 검토 후 진행하세요.

베타 단계라 한계가 있습니다:
- 양자·이혼·재혼·동시사망·태아 — 미지원
- 4촌 이상 방계 — 입력은 가능하나 자동 분석 단순화
- 부모 일부 사망 + 일부 포기 같은 복합 케이스 — 변호사 검토 필수

## 기술 스택

- 단일 HTML + Tailwind CDN + 순수 JS + SVG
- 외부 서버 의존 X (브라우저 100%)
- localStorage 저장 (브라우저 닫아도 유지)
