기능 ①

상태 일괄 입력

여러 사람의 상태(상속포기 등)를 한 번에 바꿉니다.

💡 예) 30명이 모두 상속포기 — 드롭다운 30번 대신 클릭 두세 번으로 끝.

1

우측 상단 "☑ 선택 적용" 버튼 클릭

☑ 선택 적용 ＋ 행 추가
2

각 행 앞에 체크박스가 나타남 → 바꿀 사람 체크

선택 항목을 상속포기 ▼ 으로
구나연 자녀
성하린 조카
구재운 부 (제외)

💡 "전체" 체크 = 모든 행 선택. 망인·생존 가족만 체크 해제하면 편해요.

3

상태 고르고 "적용" → 선택한 행만 한 번에 변경

상속포기 ▼ (으)로 적용 2명 선택

✅ 적용하면 체크박스가 사라지고 원래 화면으로 돌아와요.

🎁 보너스 — 행 추가 시 자동 복사

첫 사람을 "상속포기"로 두고 "＋ 행 추가"를 누르면, 새 행이 자동으로 상속포기로 추가돼요. 같은 상태가 연속이면 이게 제일 빨라요.

기능 ②

박스 색상 바꾸기

가계도 박스 색을 분류별로 직접 정합니다.

💡 예) 망인과 청구인이 둘 다 붉은 계열이라 헷갈릴 때 → 망인을 노랑으로 바꿔 구분.

1

가계도 아래 "🎨 박스 색상 바꾸기" 펼치기

▸ 🎨 박스 색상 바꾸기
2

분류별 색깔 칸을 클릭해서 원하는 색 선택 → 즉시 반영

피상속인 현 상속인 한정승인 상속포기 사망 가족 일반

색칸을 누르면 색상 선택창이 떠요. 고르면 가계도 박스가 바로 그 색으로 바뀝니다.

3

되돌리려면 "기본값 복원" 클릭

기본값 복원

바꾼 색은 자동 저장돼요. 다음에 열어도 그대로 유지됩니다.

📌 박스 색깔만 바뀌고, 좌상단 ✝(망인)·★(상속인)·故(사망) 배지는 의미 표시라 그대로예요.
